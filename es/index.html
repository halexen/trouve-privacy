Política de privacidad

Privacy Policy · Politique de confidentialité

Última actualización: abril 2026
🔒
Principio fundamental
Tus documentos originales (PDF, imágenes) permanecen almacenados localmente en tu dispositivo. Los archivos originales no se envían a nuestros servidores.
✓
Tus archivos PDF e imágenes nunca salen de tu teléfono
✓
Solo el texto extraído puede procesarse para búsqueda, indexación y análisis
✓
Sin publicidad, sin reventa de datos
✓
La copia de seguridad automática de Android está desactivada para proteger tus datos
📄
Datos recopilados
  • Contenido de documentos — el texto extraído de tus PDFs e imágenes escaneadas puede enviarse a la API Claude (Anthropic) para indexación, búsqueda, categorización y funciones inteligentes. Los archivos originales nunca se transmiten.
  • Audio de voz — cuando usas la búsqueda por voz, tu voz se procesa localmente en tu dispositivo. Nada se envía a nuestros servidores.
  • Identificador anónimo del dispositivo — usado únicamente para autenticar tus solicitudes a la API. No está vinculado a tu identidad.
  • Datos de suscripción — gestionados por RevenueCat y Apple/Google según sus propias políticas de privacidad.
📷
Permisos utilizados
  • Cámara — solo para escanear documentos en papel
  • Micrófono — solo para la búsqueda por voz
  • Archivos / Fotos — para importar tus PDFs e imágenes
  • Notificaciones — para recordatorios inteligentes (opcional)

Ningún permiso se usa para fines distintos a los descritos anteriormente.

🔔
Recordatorios inteligentes

Trouvé puede analizar el contenido de tus documentos para detectar fechas importantes (citas, plazos, facturas) y enviarte recordatorios. Las notificaciones se generan en tu dispositivo. Dependiendo de la función, el texto extraído puede procesarse para identificar información útil.

🤝
Servicios de terceros
  • Anthropic (API Claude) — procesamiento del texto extraído para indexación, búsqueda, resúmenes y chat IA. Política: anthropic.com/privacy
  • RevenueCat — gestión de suscripciones. Política: revenuecat.com/privacy
  • Apple / Google — distribución y pagos

No vendemos, alquilamos ni compartimos tus datos con anunciantes. No almacenamos copias de tus documentos en nuestros servidores.

💾
Almacenamiento y eliminación

Tus documentos se almacenan localmente en tu dispositivo. La copia de seguridad automática via iCloud (iOS) o Google Drive (Android) puede estar activa según la configuración de tu sistema.

Si desinstalas la app, todos los datos locales se eliminan permanentemente.
