1. Introducción

Trouvé es una aplicación móvil Android e iOS para buscar documentos PDF por palabras clave con resúmenes de inteligencia artificial. Esta política explica cómo recopilamos, usamos y protegemos tus datos.

2. Datos recopilados

  • Documentos PDF e imágenes importados voluntariamente
  • Texto extraído de tus documentos para la búsqueda
  • Preferencias de la app (idioma, modo oscuro)
  • Documentos vistos recientemente

3. Uso de datos

  • Extraer texto mediante la API Claude (Anthropic)
  • Permitir la búsqueda por palabras clave
  • Generar resúmenes inteligentes
  • Recordar tus preferencias

4. Compartir datos

Tus documentos se envían a la API Claude de Anthropic para la extracción de texto. Anthropic procesa estos datos según su política de privacidad. No vendemos tus datos personales.

5. Almacenamiento

Todos tus datos se almacenan localmente en tu dispositivo. No usamos servidores para almacenar tus documentos personales.

6. Seguridad

  • Datos almacenados localmente en tu dispositivo
  • Comunicaciones cifradas mediante HTTPS
  • No tenemos acceso a tus documentos personales

7. Tus derechos

  • Eliminar tus documentos desde Ajustes
  • Desinstalar la app para eliminar todos los datos
  • Contactar con soporte para cualquier pregunta

8. Menores

La aplicación no está destinada a menores de 13 años.

9. Cambios

Podemos actualizar esta política en cualquier momento. Los cambios se publicarán en esta página.
