Principio fundamental

Trouvé está diseñado para respetar tu privacidad. Tus documentos originales (PDF, imágenes) permanecen almacenados localmente en tu dispositivo. Los archivos originales no se envían a nuestros servidores.

✓ Tus archivos permanecen en tu dispositivo — algunas funciones pueden usar texto extraído para permitir la búsqueda, indexación y análisis.
📄
Datos recopilados

Trouvé solo procesa los datos necesarios para el funcionamiento de la app:

  • Contenido de documentos — el texto extraído de tus PDFs e imágenes escaneadas puede enviarse a la API Claude (Anthropic) para indexación, búsqueda, categorización y funciones inteligentes. Los archivos originales (PDFs, imágenes) nunca se transmiten.
  • Audio de voz — cuando usas la búsqueda por voz, tu voz se procesa localmente en tu dispositivo.
  • Identificador del dispositivo — se usa un identificador anónimo para autenticar tus solicitudes. No está vinculado a tu identidad.
  • Datos de suscripción — gestionados por RevenueCat y Apple/Google.
📷
Permisos utilizados
  • Cámara — para escanear documentos
  • Micrófono — búsqueda por voz
  • Archivos / Fotos — importar documentos
  • Notificaciones — recordatorios (opcional)

Ningún permiso se utiliza con otros fines.

🔔
Recordatorios inteligentes

Trouvé puede analizar el contenido de tus documentos para detectar fechas importantes y enviarte recordatorios. Las notificaciones se generan en tu dispositivo. Dependiendo de la función, el texto extraído puede procesarse para identificar información útil.

🤝
Servicios de terceros
  • Anthropic (Claude API) — procesamiento del texto extraído
  • RevenueCat — gestión de suscripciones
  • Apple / Google — pagos y distribución

No vendemos ni compartimos tus datos con fines publicitarios. No almacenamos copias de tus documentos en nuestros servidores.

💾
Almacenamiento y eliminación

Tus documentos permanecen en tu dispositivo. Si desinstalas la app, todos los datos se eliminan.
