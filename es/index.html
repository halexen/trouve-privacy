Política de privacidad

Última actualización: abril 2026

Principio fundamental

Tus documentos permanecen en tu dispositivo.

Los archivos originales (PDF, imágenes) nunca se envían a nuestros servidores.

Algunas funciones requieren procesar el texto extraído mediante un servicio de IA (Anthropic — Claude) para permitir la búsqueda, categorización y análisis.

Datos tratados

  • Texto extraído — enviado temporalmente a Claude para indexación y búsqueda
  • Documentos — almacenados únicamente en tu dispositivo
  • ID anónimo — utilizado para asegurar las solicitudes API
  • Datos de suscripción — gestionados por Apple / Google / RevenueCat

Almacenamiento

No almacenamos documentos ni contenido en nuestros servidores. Todos los datos permanecen en tu dispositivo.

Servicios externos

  • Anthropic (Claude API) — procesamiento de texto
  • RevenueCat — gestión de suscripciones
  • Apple / Google — pagos y distribución
