1. Introduction

Trouvé est une application mobile Android et iOS de recherche de documents PDF par mots-clés avec résumé par intelligence artificielle. Cette politique explique comment nous collectons, utilisons et protégeons vos données.

2. Données collectées

  • Documents PDF et images importés volontairement
  • Texte extrait de vos documents pour la recherche
  • Préférences de l'application (langue, mode sombre)
  • Documents récents consultés

3. Utilisation des données

  • Extraire le texte via l'API Claude (Anthropic)
  • Permettre la recherche par mots-clés
  • Générer des résumés intelligents
  • Mémoriser vos préférences

4. Partage des données

Vos documents sont envoyés à l'API Claude d'Anthropic pour l'extraction de texte. Anthropic traite ces données selon sa politique de confidentialité. Nous ne vendons pas vos données personnelles.

5. Stockage

Toutes vos données sont stockées localement sur votre appareil. Nous n'utilisons pas de serveurs pour stocker vos documents personnels.

6. Sécurité

  • Données stockées localement sur votre appareil
  • Communications chiffrées via HTTPS
  • Nous n'avons pas accès à vos documents personnels

7. Vos droits

  • Supprimer vos documents depuis les Réglages
  • Désinstaller l'app pour supprimer toutes les données
  • Contacter notre support pour toute question

8. Enfants

L'application n'est pas destinée aux enfants de moins de 13 ans.

9. Modifications

Nous pouvons modifier cette politique à tout moment. Les modifications seront publiées sur cette page.
